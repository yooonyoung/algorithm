1. 구구단 https://www.acmicpc.net/problem/2739

   ```
   n = int(input())
   for i in range(1,10):
       print(n, "*", i, "=", n*i)
   ```

2. A + B -3 https://www.acmicpc.net/problem/10950

   ```
   n = int(input())
   for i in range(0,n):
       a,b = input().split()
       print(int(a)+int(b))
   ```

3. 합 https://www.acmicpc.net/problem/8393

   ```
   n = int(input())
   sum = 0
   for i in range(1,n+1):
       sum += i
   print(sum)
   ```
