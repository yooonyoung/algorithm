1. 피보나치 함수 https://www.acmicpc.net/problem/1003

   ```
   t = int(input())
   while t:
       d0 = [1,0]
       d1 = [0,1]
       n = int(input())
       if n >= 2:
           for i in range(2, n+1):
               d0.append(d0[i-1]+d0[i-2])
               d1.append(d1[i-1]+d1[i-2])
       print(d0[n], d1[n])
       t -= 1
   ```
