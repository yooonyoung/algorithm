문제 링크 : https://www.acmicpc.net/problem/2294

```
n, k = map(int, input().split())
c = []
dp = [0 for i in range(k + 1)]
for i in range(n):
    c.append(int(input()))
for i in range(1, k + 1):
    a = []
    for j in c:
        if j <= i and dp[i - j] != -1:
            a.append(dp[i - j])
    if not a:
        dp[i] = -1
    else:
        dp[i] = min(a) + 1
print(dp[k])
```
